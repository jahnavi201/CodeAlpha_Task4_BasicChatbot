# 🤖 Basic Chatbot

This project is a simple **rule-based chatbot** developed using **Python** as part of my **CodeAlpha Internship**.

## 📌 Project Description

The chatbot interacts with users by responding to predefined messages such as greetings, questions, and farewells. It uses simple conditional statements to generate responses.

## 🚀 Features

- Greets the user
- Responds to common questions
- Simple rule-based conversation
- Easy-to-use console interface
- Exit option to end the chat

## 🛠️ Technologies Used

- Python
- Functions
- If-elif Statements
- Loops
- Input/Output

## ▶️ How to Run

1. Install Python.
2. Download or clone this repository.
3. Open the project folder in VS Code or Terminal.
4. Run the program:

```bash
python chatbot.py
```
